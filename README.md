# AI 投放&内容小助手（CTM Demo）

一个面向中小商家的 **AI 增长策略 + 投放建议 + 素材生成 + 复盘闭环** 小工具 Demo。  
输入行业/地区/预算/卖点后，自动输出：**用户画像（Personas）→ 渠道与预算分配 → 标题/文案/口播脚本 → 指标口径 → 迭代建议（可执行动作）**，并支持 **一键导出 JSON**，便于汇报与复用。

> 作品定位：适用于 **百度 CTM 管培生**（商业/产品/客户解决方案方向）的小型作品展示，强调“AI + 业务落地 + 结构化输出 + 闭环”。

## 核心亮点
1. **结构化输出**：所有结果按固定结构输出（JSON），方便前端渲染、复盘、二次加工。
2. **业务闭环**：策略 → 素材 → 指标 → 迭代动作，避免“只生成文案不落地”。
3. **可解释**：每个渠道给出定向与沟通角度，便于投手/运营直接执行。
4. **合规意识**：内置风险提示（夸大宣传、误导价格、资质/版权等），贴近真实投放场景。
5. **可扩展 AI**：当前为离线规则兜底版，后续可无缝接入文心/通义/智谱等大模型做增强（同一份 JSON Schema）。

## 页面/功能说明
### 1）输入信息
- 行业、城市、预算、产品/价格、卖点、目标、限制条件等

### 2）自动生成输出
- **Personas（≥3 类）**：是谁、痛点、触发场景  
- **Positioning**：核心价值、证据点、差异化、主推钩子、语气、why now  
- **Channel Plan（≥3 渠道）**：预算占比（合计 100%）、定向、沟通角度  
- **Creatives**：标题（≥10）、广告文案（≥5）、短视频口播脚本（≥3）  
- **Metrics**：CTR / CVR / CPA / ROI 口径  
- **Iteration Suggestions（≥8 条）**：基于常见问题给出可执行优化动作  
- **导出 JSON**：可复制/下载，便于面试展示细节
